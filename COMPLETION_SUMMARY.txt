================================================================================
                            QUICK START GUIDE
================================================================================

INSTALLATION:
  cd backend
  npm install
  npm run start:dev

API BASE URL:
  http://localhost:3000

EXAMPLE REQUESTS:

  Search Papers:
    GET /search?q=machine+learning&author=LeCun

  Get Paper by DOI:
    GET /papers/doi/10.1234/example

  Get Citation Network:
    GET /citations/:paperId/network?depth=2

  Find Author by ORCID:
    GET /authors/orcid/0000-0001-2345-6789

  Get Trending Papers:
    GET /analytics/trending/24h?limit=10
